# Boutique Concept

A luxury-themed frontend template for boutique rental/retail websites.

## Status
🚧 **Work in Progress** — This is a demonstration template showcasing frontend architecture and design system. Not production-ready.

## Features
- React/Next.js components
- Luxury design system (gold/cream palette)
- E-commerce components (calendar, product cards, search)
- Mobile-first responsive design
- Chat widget integration

## Tech Stack
- React/Next.js
- CSS Modules
- Responsive design utilities
